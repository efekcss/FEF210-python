# FEF210 Python Ders Notları ile Python Temeli

FEF210 Murat hocanın python notları python temeli için hatta advance'e de biraz değicek şekilde güzel bir iskelet, ana şablon sunuyor.

Bu notlardan faydalanarak Python temellerini attıktan sonra da geniş başlıkta konseptleri de öğrenmek için [roadmap.sh](https://roadmap.sh/python) sitesinden de konu takibi yaparak arta kalanları da [Fluent Python-Luciano Ramalho](fluent-python-luciano-ramalho.pdf) kitabı üzerinden tamamlayacağım. Hocanın ders notları bittiğinde de zaten bu kitabın üzerinden derinlemesine olmasa da hızlıca yeniden geçeceğim. Bu şekilde gerkli python seviyesine ulaştıktan sonra numpy, pandas, scikit-learn, PyTorch, TensorFlow gibi alanım için gerekli kütüphaneleri öğreneceğim. Bunları da yine repoda bulunan diğer iki kitap olan; [Python for Data Analysis-Wes Mckinney](Python-for-Data-Analysis-wes-mckinney.pdf) , [Hands On Machine Learning With Scikit-Learn, Keras and TensorFlow-Aurelien-Geron](Hands-On_Machine_Learning_with_Scikit-Learn-Keras-and-TensorFlow-2nd-Edition-Aurelien-Geron.pdf) kitaplarından öğreneceğim. (NOT:Telif sebebiyle kitapları repoda açıkça paylaşmayacağım.)
